# APP Te Lo Compro

**Project setup**

1. Instalar npm para generar los node_modules
```
npm install
```
2. Ejecutar el script para construir la aplicación
```
npm run build
```
3. Instalar un servidor virtual
```
npm install serve
```
4. Ejecutar la aplicación en el servidor virtual
```
serve -s dist
```
5. Se accede a la aplicación
```
http://localhost:5000
```
